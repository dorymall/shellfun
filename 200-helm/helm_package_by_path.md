# helm_package_by_path

## Function Call:

```bash
helm_package_by_path ./EmailApp
```

## Description

Packages a chart into a versioned chart archive file.

## Actual call

```bash
helm package ./EmailApp
```

## output:

```
Successfully packaged chart and saved it to: ./EmailApp-0.1.0.tgz
```

## Output files

```
EmailApp-0.1.0.tgz
```
